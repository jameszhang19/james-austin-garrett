# james-austin-garrett

The Hidden Health Habit Checker is a simple, interactive quiz tool built on Code.org’s platform that helps users quickly assess their daily health habits. As users answer “Yes” or “No” to basic questions about routines like eating breakfast, eating slowly, maintaining good posture, and exercising outdoors, the program tracks their score and builds a personalized list of small, actionable improvements based on their responses. It turns everyday health awareness into an engaging, beginner-friendly experience, offering clear, tailored tips to help users build healthier habits.
